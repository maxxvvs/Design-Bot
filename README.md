### ☕ Usage  
- #### 💻 Downloading
     ```
    >> install it on your desktop
    >> install nodejs on your pc
    >> npm init
    >> npm install discord.js @discordjs/builders dotenv
    ```
- #### 🖥️ Starting
      1 - Run In CMD npm init
      2 - Than npm install discord.js @discordjs@/builders dotenv
      2 - node index.js
